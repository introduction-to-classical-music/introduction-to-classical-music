# 0.1.0 产物清单

本文件记录 `v0.1.0` 当前公开发布矩阵中的产物边界。

## GitHub Release 公开资产

### Windows 安装版

- `Setup.0.1.0.exe`

用途：

- 面向普通用户的标准安装方式
- 安装后进入桌面启动器
- 自带默认空资料库与《不全书使用手册》

### Windows 便携版

- `BuQuanShu-Portable-0.1.0.exe`
- `BuQuanShu-Portable-0.1.0.zip`

用途：

- 面向试用、演示和无需安装的场景
- 打开后直接进入站点界面
- 内容基于发布者个人资料库站点构建

### GitHub 自动源码包

- `Source code (zip)`
- `Source code (tar.gz)`

用途：

- 供用户下载源码、离线归档或自行构建

## 公共源码仓库内容

公共源码仓库保留：

- 桌面应用源码
- 维护工具源码
- 版本检索工具源码
- 默认空资料库模板
- 《不全书使用手册》素材
- 测试、脚本与发布文档

公共源码仓库不保留：

- 开发者个人资料库原始数据
- 便携版二进制
- 互联网部署站点仓库
- 构建缓存和临时产物

## 默认资料库

公开发布的默认资料库固定为：

- 空库结构
- 《不全书使用手册》专栏

不包含：

- 测试条目
- 开发者个人资料库内容
- 来历不明的音视频或封面资源

## 额外说明

- 安装版、便携版和互联网部署站点采用分离工作区维护
- 安装版发布面是干净源码 + 默认空资料库
- 便携版发布面只有二进制资产，不建立独立 Git 仓库
- 互联网部署站点在独立部署仓库中维护
