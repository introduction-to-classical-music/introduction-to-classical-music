# 发布文档索引

本目录保存公开发布、安装、边界与发版流程相关文档。

## 当前发布矩阵

当前 `v0.1.0` 公开发布包括：

- Windows 安装版：`Setup.0.1.0.exe`
- Windows 便携版：
  - `BuQuanShu-Portable-0.1.0.exe`
  - `BuQuanShu-Portable-0.1.0.zip`
- GitHub 源码仓库：
  - `git clone`
  - GitHub 自动生成的 `Source code (zip)` 与 `Source code (tar.gz)`

默认资料库始终是：

- 空库结构
- 《不全书使用手册》专栏

不包含测试条目和开发者个人资料库内容。
