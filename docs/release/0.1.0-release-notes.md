# 不全书 0.1.0 发布说明

`0.1.0` 是不全书的首个公开版本。

本版本围绕“本地维护古典音乐资料，并构建可浏览的静态站点”这一核心目标发布，提供安装版、源码仓库和便携版三种获取方式。

## 本次发布包含什么

### Windows 安装版

- 发布资产名称：`Setup.0.1.0.exe`
- 适合希望直接安装并长期使用的普通用户
- 安装后进入桌面启动器
- 默认附带空资料库模板与《不全书使用手册》

### GitHub 源码仓库

- 适合希望自己构建、调试或参与开发的用户
- 可直接 `git clone`
- 也可使用 GitHub 自动提供的：
  - `Source code (zip)`
  - `Source code (tar.gz)`

### Windows 便携版

- 发布资产名称：
  - `BuQuanShu-Portable-0.1.0.exe`
  - `BuQuanShu-Portable-0.1.0.zip`
- 适合临时试用、演示或不希望执行安装流程的用户
- 打开后直接进入站点界面

## 本版本支持的能力

- 维护作曲家、人物、团体、作品、版本与专栏
- 构建并浏览本地古典音乐静态站点
- 在桌面启动器中打开站点、维护工具与版本检索工具
- 在版本条目中维护外部资源链接与本地资源链接

## 默认资料库边界

公开发布的默认资料库固定为：

- 空库结构
- 《不全书使用手册》专栏

不包含：

- 测试条目
- 开发者个人资料库
- 来历不明的音视频或封面资源

## 适用平台

当前公开支持平台：

- `Windows 10 x64`
- `Windows 11 x64`

## 已知边界

- 当前版本不提供应用内自动更新
- 本地资源链接通常只在当前设备有效
- 安装版、便携版和互联网部署站点采用分离工作区维护
- 互联网公开部署仍需后续完成域名、备案、OSS 与 CDN 配置
