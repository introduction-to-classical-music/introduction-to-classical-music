# 0.1.0 安装与获取说明

本版本当前提供三种获取方式：

- Windows 安装版
- GitHub 源码安装
- Windows 便携版

## 1. Windows 安装版

发布资产：

- `Setup.0.1.0.exe`

适用人群：

- 希望直接安装并长期使用的普通用户

安装步骤：

1. 打开 GitHub [Releases](https://github.com/Hektor1277/introduction-to-classical-music/releases)
2. 下载 `Setup.0.1.0.exe`
3. 双击运行安装程序
4. 选择安装目录并完成安装
5. 从开始菜单或桌面启动 `不全书`

安装版特点：

- 支持自定义安装路径
- 默认安装目录内自带空资料库模板
- 支持同版本重装和后续覆盖升级
- 启动后进入桌面启动器

卸载说明：

- 卸载会删除程序目录及其内置默认资料库
- `%APPDATA%\Introduction to Classical Music` 中的日志与运行设置默认保留

## 2. GitHub 源码安装

适用人群：

- 希望自己构建、调试或参与开发的用户

环境要求：

- `Windows 10/11 x64`
- `Node.js 22`
- `npm`
- `Python 3.13`

执行步骤：

```powershell
git clone https://github.com/Hektor1277/introduction-to-classical-music.git
cd introduction-to-classical-music
npm run bootstrap:windows
npm run doctor:windows
npm run check
npm run build
```

如需自行打包安装版：

```powershell
npm run package:windows
```

## 3. Windows 便携版

发布资产：

- `BuQuanShu-Portable-0.1.0.exe`
- `BuQuanShu-Portable-0.1.0.zip`

适用人群：

- 想快速试用、演示，或不希望执行安装流程的用户

使用方式：

1. 打开 GitHub [Releases](https://github.com/Hektor1277/introduction-to-classical-music/releases)
2. 下载便携版 `EXE` 或 `ZIP`
3. 双击运行，或解压 `ZIP` 后运行内部程序

便携版特点：

- 打开后直接进入站点界面
- 单文件 `EXE` 首次启动较慢，因为需要先自解压
- `ZIP` 解压后运行通常更快

## 默认资料库说明

安装版与源码默认构建都附带相同的默认资料库边界：

- 空库结构
- 《不全书使用手册》专栏

不包含：

- 测试条目
- 开发者个人资料库

## 边界提示

- 安装版是面向普通用户的主发布形式
- 便携版面向试用和演示，不替代安装版
- 互联网公开部署站点在独立部署仓库维护，不在本仓库内直接完成
